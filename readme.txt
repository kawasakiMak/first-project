Инструкция по работе с Git


Итак , сначало необходимо скачать *гит баш* если у вас виндовс или вы можете пользоваться своим терминалом если у вас операционная система Мак или Линукс. После вы должны будете файл который будет вашим репозиторием , можно будет выбрать любой файл. После того как выбрете вы заходите в путь вашего выбранного файла , командой ch(change directory ) path(absolute path). После этого инициализируете его командой git init. Поздравляю, вы создали ваш репозиторий!
